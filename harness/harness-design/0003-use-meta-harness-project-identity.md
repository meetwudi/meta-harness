# 0003: Use Meta Harness Project Identity

## Status

Accepted

## Context

The repository is intended to be the Meta Harness project rather than Project Bee.

Repository identity: [meetwudi/meta-harness](https://github.com/meetwudi/meta-harness)

## Decision

Use "Meta Harness" as the project identity in current harness docs.

Use `meta-harness-design` as the harness design sequence name.

## Consequences

The harness docs now describe this repository as a reusable meta-harness project.

Project-specific references should be avoided in reconstructable harness decisions unless the decision is explicitly about project identity.
