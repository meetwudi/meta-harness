# Library Discovery

This folder defines Library discovery for this Meta Harness repository.

Read [LIBRARIES.toml](LIBRARIES.toml) for checked-in Libraries.

If `LIBRARIES.local.toml` exists, read it after `LIBRARIES.toml`. It is ignored and may list private, local, machine-specific, or otherwise uncommitted Libraries.

## Index Rules

Each Library points to a place. Explore that place using its own docs and files.

Local index entries must not weaken checked-in governance.
