# Product Context

Source: human request on 2026-06-25.

PROJ-Quartz is a customer acquisition application.

It uses Meta Harness and the Knowledge Agent to drive customer acquisition
discovery and engagement by treating knowledge as code. Customer information is
intended to become part of the project knowledge.

The first version should stay simple: use CopilotKit as the chat-oriented front
end framework and let a user talk directly with the Meta Harness Knowledge
Agent. Other customer acquisition surfaces can remain empty until a human asks
to add them.

Production project knowledge should move toward database-backed storage rather
than relying on local filesystem Libraries. The specific database is not chosen
yet.
